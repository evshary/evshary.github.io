解法

可以使用gcc的特殊option

  • gcc -M xxx.c: 找出所有相依檔

  • gcc -MM xxx.c: 同-M，但不含系統檔

  • gcc -MP -MM xxx.c: 會加上其他header，避免某些compiler error

  • -MF file: 輸出的dependency檔案名

  • -MD: 同-M -MF
  • -MMD: 同-MM -MF
  • -MT: 可以更改dependency檔案內的目標，可參考关于 gcc MT MF[转]

因此只要把Makefile改成這樣即可
